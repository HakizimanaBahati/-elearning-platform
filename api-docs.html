Authentication Levels

LevelDescription
PublicNo authentication required
ProtectedRequires valid session cookie
AdminRequires admin role + valid session

HTTP Status Codes

CodeMeaning
200Success
400Bad Request - Invalid input
401Unauthorized - Invalid credentials
403Forbidden - Insufficient permissions
404Not Found
409Conflict - Resource already exists

Quick Examples

cURL

# Login
curl -X POST http://localhost:3000/api/trpc/auth.login \
  -H "Content-Type: application/json" \
  -d '{"json": {"email": "user@example.com", "password": "password123"}}'

# Get courses (public)
curl http://localhost:3000/api/trpc/courses.list

# Create course (authenticated)
curl -X POST http://localhost:3000/api/trpc/courses.create \
  -H "Content-Type: application/json" \
  -b "session_token=your_token" \
  -d '{"json": {"title": "My Course"}}'

JavaScript/TypeScript

import { createTRPCProxyClient, httpBatchLink } from '@trpc/client';
import type { AppRouter } from './server/routers';

const trpc = createTRPCProxyClient({
  links: [httpBatchLink({ url: 'http://localhost:3000/api/trpc' })],
});

const courses = await trpc.courses.list.query();
const user = await trpc.auth.login.mutate({
  email: 'user@example.com',
  password: 'password123'
});

system

Query Public system.health

Check API health status

Input

{
  timestamp: number  // Client timestamp (cannot be negative)
}

Output

{ ok: boolean }

Example: /api/trpc/system.health?input={"timestamp":1699900000}

Mutation Admin system.notifyOwner

Send notification to platform owner

Input

{
  title: string    // Min 1 character
  content: string  // Min 1 character
}

Output

{ success: boolean }

auth

Query Public auth.me

Get current authenticated user

Output

User object or null
Mutation Public auth.logout

Log out current user

Output

{ success: true }
Mutation Public auth.login

Authenticate user and create session

Input

{
  email: string    // Valid email format
  password: string
}

Output

User object
Errors: UNAUTHORIZED - Invalid credentials
Mutation Public auth.register

Register new user account

Input

{
  email: string
  password: string  // Min 6 characters
  name: string     // Min 2 characters
  role?: "user" | "instructor" | "admin"  // Default: "user"
}

Output

User object
Errors: CONFLICT - Email already exists
Mutation Protected auth.updateProfile

Update current user's profile

Input

{
  name?: string  // Min 2 characters
}

Output

Updated User object
